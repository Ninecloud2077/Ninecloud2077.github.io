如何鉴定网络热门编程视频

与营销号斗智斗智

    营销号诸位都不陌生。编程界的营销号常常利用代码博人眼球，但这些代码实际上并无技术含量，有些甚至并不能运行，因为需要其它特殊的操作。

    判断营销号的用具

    营销号常使用几种简单的语言来达到目的：
    • Python：简单易上手，第三方库功能强大，在少儿编程等各种领域都极受欢迎。
    • Windows命令行：Windows系统基本都支持，无需进行编译、安装依赖等复杂操作，双击即可运行。（注：下文有时将其简称为Cmd）
    • HTML：有浏览器（即使是IE）即可运行。难度相对较高，同时代码量看起来也比较多。
    如何判断营销号使用哪种语言？
    • 看后缀名：Python文件的后缀是.py，Windows Cmd是.bat，HTML是.html或.htm。

    为何“x行代码做出xxx”多数是作假？

    拿上面的几种语言，尝试完成3D效果的话：
    • Python很难完成3D效果，多是2D几何绘图（Turtle）。
    • Cmd基本不可能独立完成图形化。大多数Cmd实现3D效果依附于HTML。
    • 一些视频中用Cmd中的start命令运行网上的HTML代码。HTML就是网页，营销号使用start命令实际上就类似于用浏览器打开网站。
    • HTML只是一个载体，想要表现特效、动画还需要CSS、Javascript。只有HTML的代码不可能达到效果。
    总而言之，绝大多数语言是可以实现效果的，但需要的代码量远不止营销号所展现的。

    题外话：用多种语言相互串通实现目的

    • 在Python中如果出现os.system，代表用Python执行了Cmd命令。
    • 上文所提到的，在Cmd中使用start命令就类似于多语言联系。start命令会打开一个网页，利用网页呈现效果。
    • 奇思妙想出一种三语言串通方法：（注：使用Python运行Cmd的start命令，以打开HTML网页）
                  import os
            os.system("start https://www.baidu.com")
