HTML

超文本标记语言，用于网页

和大多数网页一样，本站的页面就是由HTML编写。除了HTML，一个完美的网页也离不开CSS与Javascript。

CSS与Javascript有什么用？

  • HTML是主体，需要通过CSS与Javascript完成样式、效果。
  • CSS控制了网页的样式，例如背景颜色、文字大小、图片的长宽等等。
  • Javascript可以简称js，在HTML网页中可以操控文本的变换、实现动画等。严格意义上，js是这三种中真正的编程语言，而不是标记语言。

HTML标准

各种浏览器对HTML的实现五花八门，因此设立了标准统一管理。
  • HTML4是最基础的标准，涵盖网页开发中绝大部分要使用的内容。主流浏览器（即使是IE）都支持HTML4。
  • HTML5是更新的标准，在4版本的基础上添加了音视频、画布等新元素。HTML5尚待完善，主流浏览器多数只支持部分HTML5。（IE9才开始支持）
  • CSS1/2提供了最基础最常用的样式，诸如颜色、字体、大小。而CSS3更新了动画、圆角、网格布局等。

CSS/Javascript的第三方库

  • jQuery是一个js库，用更少的代码达到更强大的效果。Bootstrap、Foundation的js项目都需要基于jQuery。
  • Bootstrap是一个前端框架，提供文字排版、颜色、布局等CSS项目，以及下拉菜单、点击按钮隐藏/显示等js项目。
  • Foundation拥有更好的js框架，例如滑块、按钮、图片缩略图。（注：如果你使用Foundation，请注意Foundation只能兼容jQuery2.x。）

题外话：网页开发中的其它进阶级语言

  • PHP可以嵌入网页实现表单（发送用户名、密码之类的消息）等操作替代js。单独作为文件的php在被访问时可以临时打印网页。
  • XML用于存储和传输数据。HTML可以从一个XML文件中读取数据并显示。
  • XHTML是一种HTML标准，结合了HTML和XML，这种标准的语法相对HTML更为严苛。
  • JSON也可以存储数据，且相对于XML更为简洁易读。绝大多数js都可以直接读取JSON。除了js，JSON也支持诸如Python的一部分编程语言。
