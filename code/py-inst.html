具体咋安装

本站可能只着重阐述Windows系统下的安装。因为Liunx之类的系统基本一句命令搞定：
sudo apt install python
在Windows上下载python，你可以去官网找下载包。
注意：如果你的电脑是Windows7，请选择3.8.10及以下的版本并且在换电脑以前打消任何更新的念头。为什么要这么强调，因为我没忍住。
下载包之后，你可以选择默认形式的安装或者自定义安装指定路径。如果你的C盘要满的话不建议用默认的，找个空的盘自定义下载。
再次注意：请把配置环境变量的勾打上（Add python 3.x to PATH），否则没有环境变量后期会很麻烦。

其它配置

你可以从windows任务栏里打开IDLE测试。这是我一开始用的法子。
但是用到后面你会发现，IDLE里编辑还是有点难搞，虽然比记事本好得多。
这时候你就可以试试万能的VSC。
关于Visual Studio Code的使用后期我会出个具体教学。
从官网下VSC安装，然后打开扩展（Ctrl + Shift + X），找到作者为Microsoft的Python扩展。
安装之后绑定下载的Python，你就可以用VSC编辑代码的同时顺畅地测试。
VSC是专业的代码编辑器，扩展装上之后可以提供代码补全、纠错等等功能，非常强大。
找到一个Python文件，VSC打开以后右键选择“Run Python File in Terminal”，等待PowerShell命令行启动就可以运行文件。
如果你没保存而选择了运行，它会自动帮你保存，非常贴心。
