简单论述这仨的区别

  • HTML：摆一个按钮在网页上。
  • CSS：规定这个按钮的形状、大小、颜色等等。
  • Javascript（一般简称js，.js做后缀的文件一般都是Javascript）：让按钮按下的时候变换一些东西。
  • HTML是网页最重要的东西，没有它CSS和js也不能用，整个网站也没有。
