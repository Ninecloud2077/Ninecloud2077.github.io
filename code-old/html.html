简单论述这仨的区别

  • HTML：摆一个按钮在网页上。
  • CSS：规定这个按钮的形状、大小、颜色等等。
  • Javascript：（一般简称js，）：让按钮按下的时候变换一些东西。
  • HTML是网页最重要的东西，没有它CSS和js也不能用，整个网站也没有。本站也使用这样的形式。
  • HTML的文件格式一般为.htm .html，CSS为.css，js为.js。

第三方库或者其它相关项目

  • CSS库Bootstrap：比较好的设计框架，导入以后声明一堆class就可以创造好看的样式，还有网格系统便于调整布局。分为2/3/4/5等版本。（本站使用4）
  • js库jQuery：大幅简化了选择HTML元素的过程。（本站使用）
  • XML：用于存储信息的语言，通过js读取在HTML上显示出来。
  • XHTML标准：XML语法十分严格，而HTML比较随意。XHTML结合两者，在HTML的基础上令语法更为严格。

其它：就讲几个经验

  • p里可以加small，可以加a，甚至可以加div，但请绝对不要p里塞p。别问咋知道的
  • 重复的style样式可以放进id或者class统一管理，重复的class可以用js（jQuery最棒）统一设置。重复class在用Bootstrap4时候常常遇到。
